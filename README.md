# RedTokyo Bot

Официальный Telegram-бот сетки чатов [@redtokyo](https://t.me/redtokyo). 

Бот разработан для автоматизации процессов в чатах сообщества, обеспечивая высокую производительность и стабильность благодаря использованию современного стека технологий на языке Rust.

## 🚀 Особенности
- **Высокая производительность**: Написан на Rust с использованием асинхронного рантайма `tokio`.
- **Надежность**: Использование современного фреймворка `telers` для работы с Telegram Bot API.
- **Масштабируемость**: Эффективная обработка обновлений, подходящая для чатов с высокой активностью.

## 🛠 Технологический стек
* **Язык**: [Rust](https://www.rust-lang.org/)
* **Runtime**: [Tokio](https://tokio.rs/)
* **Фреймворк**: [Telers](https://github.com/Desiders/telers)
